# python_course_project
simple python project
## project details
Sono partita creando la classe Studente contenente come attributi: "name", "surname", "age", "matricola" e una lista alla quale andranno aggiunti i voti.
Ho proceduto definendo delle funzioni:
la prima "presentati", stampa gli attributi della classe;
la seconda "aggiungi_voto", aggiunge il voto ad una lista se esso è compreso tra 18 e 30, stampando una conferma, altrimenti se minore di 18 stampa "voto non superato", se invece è maggiore di 30 stampa un messaggio di errore;
la terza "calcola_media", calcola la media dei voti aggiunti alla lista;
la quarta "ore_studio", stampa un messaggio con le ore di studio che vengono passate alla funzione.
Sono passata poi alla creazione di due oggetti appartenenti alla classe Studente, sui quali ho applicato le funzioni precedentemente menzionate.
